# Keil Demos
The purpose of this repository is to provide easy access to working example projects for various development boards. The projects aim to demonstrate the latest features of Arm Keil MDK.
Another aspect of having this repository is to test drive the new development flow that allows you to use local repositories directly in Keil MDK and be able to create pull requests for public repositories.
